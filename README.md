# 🎬 Netflix Clone (HTML, CSS & JavaScript)

## 🚀 Overview

This project is a **Netflix-inspired web application** built using pure **HTML, CSS, and JavaScript**. It focuses on creating a visually engaging UI with smooth animations and interactive features — without using any frameworks.

---

## 💡 Features

* 🎥 Cinematic Hero Section
* ✨ Smooth Intro Animation (Netflix-style)
* 🎬 Interactive Video Overlay (Play Button)
* 🖱️ Hover Effects on Movie Cards
* 📱 Fully Responsive Layout
* 🎨 Clean & Modern UI Design

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Bootstrap (for layout)

---
